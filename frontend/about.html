📖 About DonateChain

A full-stack decentralized donation system built for transparency and trust

🎯 Project Overview

DonateChain is a Decentralized Donation System that records every donation immutably on the Ethereum blockchain. Traditional donation platforms rely on centralized databases that can be tampered with. With smart contracts:

  • Donations are permanently recorded — no admin can alter them
  • Fund releases require DAO community votes — not a single admin's decision
  • Anyone can audit the entire history on Etherscan without trusting anyone
  • Campaign documents are stored on IPFS — tamper-proof and globally accessible

🔒 Security Measures

✅ Reentrancy Guard Mutex lock (_locked) prevents re-entrant calls
✅ Checks-Effects-Interactions State updated before external calls in releaseFunds()
✅ Access Control onlyOwner and onlyOwnerOrDAO modifiers
✅ Overflow Protection Solidity ^0.8.20 has built-in safe math
✅ Input Validation require() checks on all function inputs
✅ Vote Snapshot Voting power frozen at proposal creation (anti-manipulation)

🚀 Quick Setup Guide

Step 1 — Install MetaMask extension from metamask.io

Step 2 — Get test ETH from sepoliafaucet.com

Step 3 — Open Remix IDE → Compile & Deploy DonationContract.sol

Step 4 — Deploy GovernanceDAO.sol (pass DonationContract address as arg)

Step 5 — On DonationContract, call setAuthorizedDAO(daoAddress)

Step 6 — Paste both addresses into frontend/js/config.js

Step 7 — Open frontend/index.html in browser → Done!
