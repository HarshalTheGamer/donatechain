ℹ️ DAO Workflow

1. Admin deploys GovernanceDAO with DonationContract address

2. Admin calls setAuthorizedDAO(daoAddress) on DonationContract

3. Admin grants governance tokens to community members

4. Any token holder creates a proposal (campaign ID + voter list)

5. Token holders vote FOR or AGAINST within 2 minutes (lab testing period)

6. After deadline, anyone calls Execute — if passed, funds auto-release to beneficiary
